# Ranabothu Manogna— Responsive Personal Portfolio

This repository contains a single-file responsive personal portfolio built with *HTML + inline CSS*.

The page is a static site (index.html) that includes:
- Header with logo/name and responsive navigation (hamburger on mobile)
- Hero with call-to-action
- Skills grid (≥6 skills)
- Projects section (example cards)
- Contact form (static) and contact info
- Footer with current year (auto-filled by small JS)

## Files in this repo

## How it was created
- Initial template: manually created (single-file) — no frameworks, only HTML/CSS/vanilla JS.
- Purpose: assignment submission — responsive layout and accessible markup.

## How to run locally
1. Clone or download this repository.
2. Open index.html in any browser (no build step required).
